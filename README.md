# Приложение для расчета бонусных миль (Java)

Учебный проект, посвященный программированию на языке Java. Основная задача — освоение работы с примитивными типами данных, понимание логики их взаимодействия, а также закрепление базовых правил чистоты кода (Clean Code) и работы с Git.

## ✈️ Суть задачи (Бизнес-логика)
Авиаперевозчики предлагают бонусные программы, начисляющие бесплатные мили за перелёты. Программа рассчитывает количество начисленных миль за купленный билет по следующему правилу:
* За каждые **20 рублей**, потраченные на билет, начисляется **1 миля**.
* Дробные мили не начисляются (округление вниз до целого числа).

**Пример работы логики:**
При стоимости билета в `13 676` рублей пассажиру должно начислиться `683` бонуса.

---

## 💻 Что было реализовано в проекте
* Создана переменная для хранения стоимости билета (тип данных выбран с учетом специфики денежных расчетов).
* Написан алгоритм вычисления итоговых бонусных миль на основе заданных условий.
* Результат расчета выводится в консоль программы.
* Код строго отформатирован в соответствии с **Java Code Conventions** (правильное именование переменных в стиле `camelCase`, отступы, структура класса).
